Sistema web inspirado no aplicativo Moovit, desenvolvido com foco em *mobilidade urbana acessível*, experiência do usuário e inclusão digital.
O projeto busca oferecer uma plataforma moderna e intuitiva para consulta de linhas, favoritos, localização em tempo real e recursos avançados de acessibilidade.

---

## 📌 Sobre o Projeto

O *MOVEASY* foi criado como uma proposta de redesign e evolução de plataformas de transporte urbano, priorizando:

* ♿ Acessibilidade digital
* 🧠 Experiência do usuário (UX/UI)
* 📱 Responsividade Mobile First
* 🚌 Consulta de linhas e ônibus
* ⭐ Sistema de favoritos
* 👤 Perfil do usuário
* 🌙 Modos visuais adaptativos
* 📍 Integração com mapas e localização

O sistema foi desenvolvido utilizando tecnologias web modernas e pensado para funcionar tanto em computadores quanto em dispositivos móveis.

---

# ✨ Funcionalidades

## 🚌 Sistema de Transporte

* Consulta de linhas de ônibus
* Visualização de rotas
* Atualização em tempo real
* Integração com mapas
* Simulação de ônibus em movimento

## 👤 Área do Usuário

* Perfil personalizado
* Sistema de favoritos
* Salvamento local com LocalStorage
* Histórico de preferências

## ♿ Acessibilidade Avançada

* Vlibras implementado
* Aumento e redução de fonte
* Máscara de leitura
* Redução de animações
* Estrutura visual organizada
* Compatibilidade com padrões WCAG

# 🚀 Objetivo

O principal objetivo do MOVEASY é demonstrar como tecnologia, acessibilidade e design podem trabalhar juntos para criar experiências digitais mais inclusivas e eficientes para mobilidade urbana.
